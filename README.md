# ESP32 Web Dashboard

This is a simple web interface for controlling LEDs on an ESP32 via MQTT.

## How to use
1. Connect the ESP32 to your network.
2. Open the webpage in a browser.
3. Click buttons to turn LEDs ON/OFF.

## Files
- `index.html` : main webpage
- `style.css`  : optional styling
- `script.js`  : optional JavaScript
